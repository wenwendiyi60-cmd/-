# 回归测试：过程证据不得替代结论证据

**测试编号**：T-05  
**测试目标**：验证口令在断语阶段单独列出结论依据，而非直接引用分析过程中的中间推论作为断语支撑  
**适用口令**：断卦引擎-2.0-Lite-Lock.md 及所有后续版本  

---

## 背景说明

"过程证据替代结论证据"是一种常见的幻觉模式：

> 分析时说"用神受克，力弱"，然后在断语中直接用这句话作为断语依据——
> 但"用神受克"是过程中的描述，不是独立的、可检验的结论依据。

合规的断语必须在第五阶段**重新明确列出**支撑结论的强/中证据，使断语可以独立被核查，不依赖阅读全部分析过程。

---

## 测试用例 T-05-A：断语依据是否独立可读

### 输入

```
本卦风水涣，三爻为动，起卦日 2026-04-07（壬子日，庚辰月），问近期求财是否顺利。
```

### 预期行为

**断语部分**必须：
- 独立列出支撑结论的具体证据（如"妻财爻卯，受月建辰土克，力弱"），而非仅写"如前分析"
- 区分强证据与中证据
- 不得仅用"综合以上分析"作为依据过渡句

**合规断语示例**：

```
【断语】
结论：事难成

强证据：
- 用神妻财爻（卯）受月建辰土直接克，入衰
- 三爻动爻（兄弟亥）克财，动爻未入空，克力有效

中证据：
- 用神未得日建生助（日建子生亥兄弟，不生财爻）

不确定项：
- 变卦中财爻状态需结合整体判断，此处仅作参考
```

**违规断语示例**：

```
【断语】
综合以上分析，用神较弱，兄弟动而克财，因此事难成。
```
（违规原因：依赖"以上分析"，断语本身无独立证据）

## 测试用例 T-05-B：仅有弱证据时不得给出强结论

### 输入

```
本卦水天需，无动爻，起卦日 2026-04-07（壬子日，庚辰月），问升职。
```

（假设：用神官鬼爻未受日月生克，也未入旬空，无动爻影响）

### 预期行为

口令必须：
- 在证据分层中说明"当前仅有弱证据，无强/中级生克信号"
- 断语输出："信息不足以支持明确结论"或"尚可，偏向平淡"
- **不得**强行给出"事成"或"事难成"的确定性结论
