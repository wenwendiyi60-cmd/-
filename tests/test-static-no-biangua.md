# 回归测试：无动爻时禁止生成变卦

**测试编号**：T-03  
**测试目标**：验证口令在用户明确说明无动爻时，完全不生成、提及或暗示变卦  
**适用口令**：断卦引擎-2.0-Lite-Lock.md 及所有后续版本  

---

## 测试用例 T-03-A：用户明确说明无动爻

### 输入

```
本卦雷地豫，无动爻，起卦日 2026-04-07（壬子日，庚辰月），问近期财运。
```

### 预期行为

**阶段1锁盘输出**中：
- 变卦字段显示："无（因无动爻）"

**阶段2分析中**：
- 完全不出现"变卦"二字
- 不出现任何"变为……"、"变卦参考……"的表达
- 阶段2第四步（动爻与变卦分析）输出："本卦无动爻，第四阶段跳过"

**断语中**：
- 不引用任何变卦信息作为依据

### 不可接受的行为

- "虽无动爻，变卦方向可参考……"
- "此卦静而有变之象……"
- 在锁盘结果中填写任何卦名作为变卦
- 在分析中提及变卦爻位

## 测试用例 T-03-B：排盘格式口令无动爻场景

### 适用口令

`paipan-format.md`

### 输入

```
本卦雷地豫，无动爻，起卦日 2026-04-07（壬子日，庚辰月）。
```

### 预期行为

排盘表格中：
- 动爻列：所有爻均标 ○（无动爻）
- 变爻列：所有爻均标 "—"
- 变卦字段：填"无"
- 排盘结束后无任何分析内容
