# 回归测试：旬空缺失时的行为

**测试编号**：T-02  
**测试目标**：验证口令在用户未提供起卦日期时，拒绝计算或列出旬空，不得推测或默认  
**适用口令**：断卦引擎-2.0-Lite-Lock.md 及所有后续版本  

---

## 测试用例 T-02-A：用户未提供起卦日期

### 输入

```
本卦地火明夷，无动爻，问事业。
```

### 预期行为

口令必须：
- 发现起卦日期缺失
- 停止，说明："起卦日期未提供，无法计算旬空和日月建"
- 询问用户提供起卦日期
- **不得**列出任何旬空地支
- **不得**推算日建或月建

### 不可接受的行为

- "当前大约是壬子旬，旬空为戌亥……"
- "今日日建参考……"
- 跳过旬空字段直接输出不完整的锁盘结果

## 测试用例 T-02-B：用户提供日期但格式不明确

### 输入

```
本卦地火明夷，无动爻，起卦日期是上周三，问事业。
```

### 预期行为

口令必须：
- 识别日期不明确（"上周三"无法确定具体干支）
- 询问用户提供具体日期（年月日）
- **不得**自行推算"上周三"对应的干支

## 测试用例 T-02-C：用户提供日期后旬空应正确计算

### 输入

```
本卦地火明夷，无动爻，起卦日期 2026-04-07（壬子日），庚辰月，问事业。
```

### 预期行为

口令必须：
- 壬子日属壬子旬，旬空为戌、亥
- 在锁盘输出中正确列出"旬空：戌、亥"
- 在阶段2分析中，检查各爻地支是否入空
