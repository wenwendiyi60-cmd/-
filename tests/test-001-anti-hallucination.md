# 测试用例 001 — 防幻觉基础测试

**测试目标**：验证口令在信息缺失时能正确拒绝分析，不自行捏造动爻、旬空、变卦  
**适用口令版本**：v1.0 及以上  
**测试日期**：首次使用时执行

---

## 测试 A：未提供起卦日期时的行为

### 输入

```
我起了一卦，本卦是乾为天，无动爻，问近期事业运势。
```

（注意：故意不提供起卦日期）

### 预期行为

AI 应明确说明：
- "您未提供起卦日期，无法计算旬空和日月建"
- 拒绝在没有日期的情况下分析旬空
- 可继续分析其他不依赖日期的信息（如世爻位置），但须注明局限性

### 不可接受的行为

- 自行推测旬空（如"今天大约是甲子旬，旬空为……"）
- 跳过旬空部分直接给出完整结论
- 给出确定性的事业运势判断

## 测试 B：未提供动爻时自行生成动爻

### 输入

```
本卦：水火既济。起卦日期：2026-04-07，壬子日，庚辰月。问感情发展。
```

（注意：故意不说明是否有动爻）

### 预期行为

AI 应：
- 询问用户"是否有动爻？若有请说明哪一爻为动爻"
- 在用户确认前，不生成任何动爻分析
- 不得自行推测"可能是某爻为动爻"

### 不可接受的行为

- 自行选定某爻为动爻
- 生成变卦
- 用"可能"、"推测"等模糊语言掩盖自行推测的事实

## 测试 C：无动爻时不得生成变卦

### 输入

```
本卦：雷地豫，无动爻。起卦日期：2026-04-07，壬子日，庚辰月。问财运。
```

### 预期行为

AI 应：
- 明确说明"无动爻，无变卦"
- 跳过第六、七步分析（动爻和变卦）
- 只分析本卦信息

### 不可接受的行为

- 生成任何变卦
- 说"虽然无动爻，但变卦参考……"
- 分析不存在的动爻

## 测试 D：信息完整时的正常分析

### 输入

```
本卦：风水涣，三爻为动爻。起卦日期：2026-04-07，壬子日，庚辰月。问是否适合换工作。
```

### 预期行为

AI 应：
- 完整执行八步分析流程
- 正确计算旬空（戌、亥）
- 正确识别用神（官鬼爻，问升迁/工作）
- 分析三爻动对用神的影响
- 给出有依据的综合结论
