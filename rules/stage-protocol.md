# 阶段协议 · Stage Protocol

> 定义断卦引擎的分阶段执行规则。所有口令必须严格遵守阶段顺序，禁止跳跃。

---

## 总体结构

```
阶段1：锁盘（盘面确认）
    ↓ 完整确认后才能进入
阶段2：分析（固定顺序）
    ├─ 2.1 用神定位
    ├─ 2.2 旺衰与力度
    ├─ 2.3 双面取象
    ├─ 2.4 证据分层
    └─ 2.5 四问裁决
    ↓
断语输出
```

---

## 阶段1：锁盘

**职责**：只收集信息，只确认盘面，不做任何解卦动作。

### 必须收集的字段

| 字段 | 说明 |
|------|------|
| 本卦卦名 | 或逐爻阴阳（初爻→上爻） |
| 动爻 | 第几爻，或明确"无动爻" |
| 起卦日期 | 年月日，用于计算旬空与日月建 |
| 所问之事 | 用于确定用神方向 |

### 锁盘输出格式

```
【阶段1 · 盘面锁定】
━━━━━━━━━━━━━━━━━━
本卦：___        变卦：___ / 无
起卦日：___      干支：___
旬空：___（依___旬计算）
日建：___        月建：___
动爻：___        用神方向：___
世爻：第___爻    应爻：第___爻
━━━━━━━━━━━━━━━━━━
盘面已锁定。进入阶段2。
```

### 阶段1禁止行为

- 禁止在锁盘输出中附加任何分析或暗示
- 禁止在信息不完整时输出锁盘结果
- 禁止在用户补充信息后不重新确认直接继续

---

## 阶段2：分析

**前提**：阶段1盘面已完整锁定。

### 2.1 用神定位

- 依据所问之事，从六亲中确定用神
- 标注用神爻位、地支、当前爻的阴阳
- 参考 `terminology.md` 中的用神速查表

### 2.2 旺衰与力度

分析用神受日建、月建的生克冲合：

```
日建 [地支] 对用神 [地支]：生 / 克 / 冲 / 合 / 无关
月建 [地支] 对用神 [地支]：生 / 克 / 冲 / 合 / 无关
旬空：用神是否入空？是 / 否
合绊：用神是否被其他爻合住？是（第___爻___合）/ 否
综合力度：旺 / 相 / 平 / 衰 / 极衰
```

每条结论必须注明依据，禁止直接给出"用神弱"而不说明原因。

### 2.3 双面取象

- 从有利方向取象：用神有哪些得力之处
- 从不利方向取象：用神有哪些受损之处
- 两面均须陈述，禁止单面取象

### 2.4 证据分层

将分析证据按强度分层：

| 层级 | 说明 |
|------|------|
| 强证据 | 日月双克、入旬空、被动爻直接克 |
| 中证据 | 单克、单冲、合绊 |
| 弱证据 | 无直接生克、仅参考变卦 |

过程中的中间推论不得直接作为断语依据（防止"过程证据替代结论证据"）。

### 2.5 四问裁决

在输出断语前，逐一回答：

1. **用神是否有力？** （旺衰判断结论）
2. **用神是否自由？** （是否被合绊、冲散）
3. **动爻是否有效？** （动爻是否入空、是否克或生用神）
4. **证据是否充分？** （是否达到强/中证据，还是只有弱证据）

四问回答完毕后，输出断语。

---

## 断语格式

```
【断语】
结论：_____
（明确选一：事成 / 事难成 / 尚可 / 信息不足无法断定）

强证据：
- ___

中证据：
- ___

不确定项：
- ___（若有）
```

---

## 阶段跳跃处理

若用户要求跳过阶段1直接分析：

```
阶段1尚未完成，无法进入分析。
请先提供：[列出缺失字段]
```

若分析中途用户补充信息：

```
收到新信息，返回阶段1重新锁盘。
```
