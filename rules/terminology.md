# 术语定义表 · Terminology

> 本文件定义本系统使用的核心六爻术语，供口令编写和案例记录统一使用。

---

## 一、基础结构术语

| 术语 | 定义 |
|------|------|
| 本卦 | 起卦时得到的原始六爻卦象 |
| 变卦 | 本卦动爻变化后形成的新卦，无动爻则无变卦 |
| 动爻 | 用户明确指定的发生变化的爻位，必须由用户提供 |
| 静爻 | 未发生变化的爻位 |
| 世爻 | 代表问卦人自身的爻位，依卦宫固定规则确定 |
| 应爻 | 与世爻相对的爻位，代表对方或事件方 |

---

## 二、时间与旺衰术语

| 术语 | 定义 |
|------|------|
| 日建 | 起卦当日的地支，对各爻有直接生克冲合影响 |
| 月建 | 起卦当月的月支（以节气为准，非公历月份） |
| 旬空 | 依起卦当日天干地支确定所在旬，旬内缺失的两个地支 |
| 出空 | 旬空地支被冲，或进入下一旬后不再空亡 |
| 旺 | 爻的地支当令（与日建或月建同气或得生）|
| 衰 | 爻的地支失令（被日建或月建克制）|
| 入空 | 爻的地支为当日旬空地支，力量大减 |
| 合绊 | 爻被其他爻地支六合，行动受限 |
| 冲 | 地支六冲，产生动荡或破坏 |

## 三、六亲术语

| 六亲 | 对应关系（以日主五行为参考） |
|------|------|
| 父母爻 | 生我者为父母 |
| 兄弟爻 | 与我同类者为兄弟 |
| 子孙爻 | 我生者为子孙 |
| 妻财爻 | 我克者为妻财 |
| 官鬼爻 | 克我者为官鬼 |

## 四、用神速查表

| 所问之事 | 用神 |
|----------|------|
| 问财运、求财、问生意 | 妻财爻 |
| 问官职、升迁、考核、官司 | 官鬼爻 |
| 问子女、求子、问学生 | 子孙爻 |
| 问文书、合同、学业、考试 | 父母爻 |
| 问兄弟、朋友、合伙人 | 兄弟爻 |
| 问婚姻（男问） | 妻财爻 |
| 问婚姻（女问） | 官鬼爻 |
| 问病（自身） | 世爻为主，官鬼为病神，子孙为药神 |
| 问失物 | 妻财爻 |
| 问出行 | 子孙爻（或世爻） |

## 五、分析链术语

| 术语 | 定义 |
|------|------|
| 旺衰与力度 | 判断用神的综合强弱，是断语的基础 |
| 双面取象 | 同时分析有利与不利两面，禁止单面取象 |
| 证据分层 | 将分析证据按强/中/弱分层，避免以弱证据做强结论 |
| 四问裁决 | 断语前的四个固定问题（有力？自由？动爻有效？证据充分？） |
| 过程证据 | 分析阶段的中间推论，不得直接作为断语依据 |
| 结论证据 | 断语阶段明确列出、支撑最终结论的强/中级证据 |
| 锁盘 | 阶段1的信息确认动作，完成前不得分析 |

## 六、系统口令术语

| 术语 | 定义 |
|------|------|
| Lite Lock | 精简锁定版口令，核心行为固化，不在日常迭代中修改 |
| 母版 | 包含完整规则的总口令，作为所有变体版本的源头 |
| 排盘格式口令 | 只输出标准排盘格式，不含断卦逻辑的独立口令 |
| 回归测试 | 用固定输入验证口令行为是否符合预期的测试用例 |
