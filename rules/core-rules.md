# 六爻分析核心规则

> 本文件定义本系统所有口令必须遵守的六爻分析规范。
> 任何口令版本不得违反本文件中的原则。

---

## 一、输入规范

### 1.1 必须由用户提供的信息

| 信息项 | 说明 |
|--------|------|
| 本卦六爻 | 从初爻到上爻，逐爻列出（阴/阳） |
| 动爻 | 哪一爻为动爻（若有）；若无动爻，明确说明 |
| 起卦日期 | 用于计算旬空、日月建 |
| 起卦时间（可选）| 用于时辰参考 |
| 问题（用神方向）| 问什么事，以确定用神 |

### 1.2 禁止推测的内容

- **禁止**在用户未提供动爻的情况下，自行判断哪爻为动爻
- **禁止**在用户未提供日期的情况下，自行列出旬空
- **禁止**在无动爻的情况下，生成变卦
- **禁止**在信息残缺时给出确定性结论

---

## 二、起卦与排盘规则

### 2.1 六亲安排

六亲（父母、兄弟、子孙、妻财、官鬼）根据卦宫和世爻所属五行，按生克关系逐爻安装，不得随意指定。

### 2.2 世爻与应爻

- 世爻位置依据该卦所属卦宫的固定规则确定
- 应爻为世爻对面固定位置（世三应六，世四应一，等）
- 不得自行调换世应爻位置

### 2.3 旬空

- 旬空必须依据起卦**当日**的天干地支，找出所在旬，确定旬空两支
- 不得默认使用固定旬空，必须依日计算
- 若用户未提供起卦日期，**明确说明无法判断旬空**

### 2.4 日月建

- 日建：起卦当日的地支
- 月建：起卦当月的月支（以节气为准）
- 用于判断爻的旺衰与冲合

---

## 三、分析流程

分析必须按以下步骤进行，**不得跳步，不得倒推**：

```
第一步：确认用神（根据问题类型）
第二步：确认世爻、应爻位置
第三步：计算旬空（依据起卦日干支）
第四步：判断用神旺衰（日月建生克冲合）
第五步：判断用神是否入旬空
第六步：分析动爻影响（若有动爻）
第七步：分析变卦（仅在有动爻时）
第八步：综合给出判断，并注明每条结论的依据
```

---

## 四、输出规范

### 4.1 每条结论必须注明依据

错误示例：
> 用神弱，事不成。

正确示例：
> 用神（妻财爻）居四爻，值日月双克，又入旬空，故判断为弱。结合问财之事，财爻过弱，事恐难成。

### 4.2 不确定时的标准表达

当信息不足以得出结论时，使用以下表达：

- "此处信息不足，无法判断。"
- "用户未提供起卦日期，旬空无法计算，暂不分析旬空影响。"
- "用户未指定动爻，本次按无动爻处理，不生成变卦。"

### 4.3 禁止使用的表达

- "可能是动爻……"（未经用户确认）
- "推测旬空为……"（未经计算）
- "变卦应为……"（无动爻时）
- "一般来说这种卦……"（脱离具体六爻信息的泛化论断）

## 五、用神速查表

| 所问之事 | 用神 |
|----------|------|
| 问财运、求财 | 妻财爻 |
| 问官职、升迁、官司 | 官鬼爻 |
| 问子女、求子 | 子孙爻 |
| 问文书、签约、学业 | 父母爻 |
| 问兄弟、朋友、合伙 | 兄弟爻 |
| 问病（自身）| 世爻为主，官鬼为病神 |
| 问婚姻（男问）| 妻财爻 |
| 问婚姻（女问）| 官鬼爻 |
