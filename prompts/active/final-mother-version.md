## 开发目标

总母版将在 Lite Lock 基础上扩展以下能力：

- [ ] 显式内嵌六十四卦卦宫归属表（减少模型排盘错误）
- [ ] 纳甲安爻规则显式化（不依赖模型内置知识）
- [ ] 支持"问病"专项分析路径
- [ ] 支持"问诉讼"专项分析路径
- [ ] 更细致的旺衰等级划分（旺 / 相 / 休 / 囚 / 死）
- [ ] 神煞规则（天乙贵人、六合、六冲等）
- [ ] 多动爻时的处理优先级规则

---

## 开发规范

1. 所有新规则必须先写入 `rules/` 对应文件，再引用到口令中
2. 每个新功能必须在 `tests/` 中配套回归测试
3. 总母版发布前需通过 `rules/audit-checklist.md` 的全部检查项
4. 发布时将本文件重命名为 `v{版本号}-final-mother.md`，归档旧版 Lite Lock
